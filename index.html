Uvod

Source kod je niz instrukcija napisanih u nekom programskom jeziku. Kada programer želi objaviti svoj software (program, aplikacije...), može odlučiti hoće li software biti otvorenog koda ili zatvorenog koda.

Open source (otvoreni kod)

Otvoreni kod je software čije licence dopuštaju korisniku da mijenja i dijeli sadržaj source koda. Dostupnost source koda omogućuje brzo rješavanje problema i unaprjeđivanje programa u suradnji s drugim programerima, što je jedna od glavnih ideja otvorenog koda.


Iako je software otvorenog koda često besplatan, za neke je aplikacije potrebno plačati pretplatu za dodatne opcije (sigurnosna ažuriranja, uklanjanje oglasa...).

Closed source (zatvoreni kod)

Za razliku od otvorenog koda, kod softwarea zatvorenog koda korisnik nema pristup source kodu. Licence softwarea zatvorenog koda ne dopuštaju korisniku dijeljenje i mijenjanje sadržaja source koda. Prilikom preuzimanja softwarea korisnik dobije samo izvršnu datoteku.
